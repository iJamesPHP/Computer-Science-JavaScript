JavaScript DOM Interaction My documentation for question 1 of the controlled assesemnt

(i) - Explain how you ran the script.

For this question I was given some example code that I needed to work out how to execute and explain how it works.
Here's the code I was given: 

<!DOCTYPE html>
<html>
  <body>
    <h1>Change an HTML element</h1>
    <p id="msg">Now you see me.</p>
    <button type="button"
      onclick="document.getElementById('msg').innerHTML = 'Gone!'">
      Click Me!
    </button>
    <button type="button"
      onclick="document.getElementById('msg').innerHTML = 'Back again!'">
      Bring me back!
    </button>
  </body>
</html>
I saw that it was HTML with a small amount of JavaScript being used in the onclick attributes of the buttons. I started by making an index.html file.

Then I pasted the code into the file.



Now that I have the index.html file with the code in it, I can open up my web browser (Safari in this case) and visit the file location to see the executed code.



Brilliant, the code has been executed, now let's see what it does by pressing the buttons on the page.


(ii) - Explain how the script works.

These buttons are able to modify the elements on the page by using JavaScript. Let's have a look at each individual line of this script.

<!DOCTYPE html>

This sets the document type. This is not required but is good practice to use as some very old browsers still use it to differentiate document types.


<html>

This is the opening HTML tag. The main contents for the document goes in here.


<body>

This is the opening body tag. The main visible contents for the document goes in here.


<h1>Change an HTML element</h1>

This is a H1 element, it is the largest pre-defined heading style in HTML. This is used to display the title of the page.


<p id="msg">Now you see me.</p>

This is a P element, it is used for text and general writing on a web page. This P element has been given an id attribute so that it can be referenced by the JavaScript. In this case it will be referenced using msg.


<button type="button" onclick="document.getElementById('msg').innerHTML = 'Gone!'"> Click Me! </button>

This is a button element, it automatically styles the contained text inside a button, in this case, the text is Click Me!. This button element has been given an onclick attribute. When the button is clicked, the script within this attribute is run, when this button is clicked, JavaScript finds the element with an ID of msg within the DOM then changes it's text to Gone!.


<button type="button" onclick="document.getElementById('msg').innerHTML = 'Back again!'"> Bring me back! </button>

This is another button element, it automatically styles the contained text inside a button, in this case, the text is Bring me back!. This button element has been given an onclick attribute. When the button is clicked, the script within this attribute is run, when this button is clicked, JavaScript finds the element with an ID of msg within the DOM then changes it's text to Back again!.


Then all the unclosed tags, in this case <html> and <body>.
